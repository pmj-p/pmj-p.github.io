Large-scale pancreatic cancer detection via non-contrast CT and deep learning

发表于 2024-06-03 分类于 parper 阅读次数： Valine：
本文字数： 1.8k 阅读时长 ≈ 2 分钟

论文题目：Large-scale pancreatic cancer detection via non-contrast CT and deep learning

来源：nature medicine 2023

摘要：

胰腺导管腺癌（PDAC）是最致命的实体恶性肿瘤，通常在晚期和不可手术的阶段被发现。早期或偶然发现与延长生存期相关，但由于假阳性的低流行率和潜在危害，使用单一检测方法筛查无症状个体的PDAC仍然不可行。

非对比计算机断层扫描（CT），常规用于临床适应症，具有大规模筛查的潜力，然而，使用非对比CT识别PDAC一直被认为是不可能的。在这里，我们开发了一种深度学习方法，即人工智能胰腺癌检测（PANDA），可以通过非对比CT检测和分类胰腺病变。

PANDA在来自单一中心的3208名患者数据集上进行训练。PANDA在涉及10个中心的6,239名患者的多中心验证中，实现了0.986至0.996的接收者操作特征曲线下面积（AUC）用于病变检测，其性能在PDAC（胰腺导管腺癌）识别的敏感性上比放射科医师的平均表现高出34.1%，在特异性上高出6.3%，并且在包含20,530名连续患者的现实世界多场景验证中，病变检测的敏感性达到92.9%，特异性达到99.9%。值得注意的是，PANDA在使用非增强CT扫描时显示出与使用增强CT扫描的放射学报告在区分常见胰腺病变亚型方面的非劣性。PANDA有可能成为大规模胰腺癌筛查的新工具。

Introduction

非对比度计算机断层扫描（CT）在资源有限地区的体检中心和医院中广泛使用。与对比增强CT（诊断胰腺导管腺癌（PDAC）的主要成像方式）相比，非对比度CT使患者承受较低的辐射剂量，并消除了对造影剂不良反应的风险。

除了在急诊科和社区医院常规使用的腹部非对比度CT外，胸部非对比度CT也可以完全或部分扫描胰腺区域，并且在多种临床场景中是最常进行的CT检查（即占所有进行的CT检查的近40%），如肺癌筛查。

尽管即使对于经验丰富的放射科医师来说，从非对比度CT中识别PDAC也具有挑战性，但最近的研究表明，人工智能（AI）在各种医学图像分析任务上可以匹敌甚至超越人类专家；此外，AI能够从常规图像中合成对比增强的医学图像。基于AI的机会性筛查通过非对比度CT有潜力在几个临床领域中推进大量无症状患者的PDAC早期检测，只需承担最小的额外成本和辐射暴露。

在这项研究中，我们展示了我们提出的人工智能方法PANDA（图1，利用人工智能检测胰腺癌），它能够在非对比度CT上以高精度检测和诊断PDAC（胰腺导管腺癌）和非PDAC（非胰腺导管腺癌）病变，并可被迅速用于大规模无症状患者群体的机会性筛查。这将安全有效地检测到被标准诊疗技术遗漏的早期恶性肿瘤，并在某些情况下能够及时进行治疗以实现治愈的意图。

我们的研究首先在腹部非对比度CT扫描上对PANDA进行内部评估，并将其性能与两项涉及48位放射科医师对非对比度和对比度CT成像的阅读研究的结果进行比较。然后，我们在大型外部多中心测试队列（n=5,337）上验证PANDA，以评估其在不同设置中的普适性。为了扩大适用的患者人群范围，我们研究了在胸部CT上应用PANDA的可行性。最后，为了验证与现实临床转化相关的关键问题，我们探索了将PANDA整合到大规模现实世界多场景的常规临床流程中，涉及来自四种设置（即体检、急诊、门诊和住院）的20,530名连续患者的腹部或胸部非对比度CT扫描。

这个深度学习模型PANDA能够检测和诊断PDAC和七个non-PDCA病变亚型，即胰腺神经内分泌肿瘤（PNET），固体假乳头状肿瘤（SPT），导管内乳头状粘液肿瘤（IPMN）、粘液性囊性肿瘤（MCN）、浆液性囊性肿瘤（SCN）、慢性胰腺炎和“其他”。这个模型可以检测胰腺病变的存在，分割病变，并对病变亚型进行分类。

PANDA由三个阶段组成的级联：

  1. nnU-Net：分割胰腺区域，并裁剪图片大小
  2. 构建CNNs和分类头来区分对比CT中病变的细微纹理变化（训练集交叉验证的病变检测达到99%特异性，减少假阳性预测）
  3. 胰腺病变的鉴别诊断，当在第二阶段发现异常，则会结合一个辅助记忆transformer来自动编码胰腺病变的特征，例如局部纹理、位置和胰腺形状，以实现更精确的细粒度分类。

三个任务上评估：

  1. 病变检测：病变与正常相比
  2. 主要诊断：PDAC与非PDAC相比
  3. 鉴别诊断：PDAC和非PDAC病变亚型的分类
